# 需求 2：UI 紧凑化 + SEO 优化 + 字幕/导图下载 — 设计方案

## 一、问题诊断

### 问题 1：页面不紧凑
当前 DownloadModal 是 `max-w-lg`（520px）的单列布局，视频信息、AI 总结（4个Tab）、画质选择、下载按钮全部纵向堆叠，用户需要大量滚动才能看到全部内容。

### 问题 2：SEO 缺失
`index.html` 仅有 `<title>` 和 `<meta viewport>`，完全缺少 description、keywords、Open Graph、Twitter Card、JSON-LD 结构化数据、canonical URL。

### 问题 3：功能缺失
- 字幕只能看不能下载
- 思维导图只能在小窗口看，不能全屏也不能下载

---

## 二、方案设计

### 需求 1：UI 紧凑化 — 双栏宽屏布局

改造 DownloadModal 为 `max-w-5xl`（1024px）双栏布局：

```
┌──────────────────────────────────────────────────────────┐
│  ┌─────────────────────┐  ┌────────────────────────────┐ │
│  │  左栏 (40%)          │  │  右栏 (60%)                │ │
│  │                     │  │                            │ │
│  │  ┌───────────────┐  │  │  📝总结  🧠导图  📋字幕  💬问答 │ │
│  │  │  缩略图封面    │  │  │  [Tab 切换栏]              │ │
│  │  └───────────────┘  │  │                            │ │
│  │                     │  │  ┌──────────────────────┐  │ │
│  │  标题               │  │  │                      │  │ │
│  │  UP主 · 数据统计     │  │  │  Tab 内容区           │  │ │
│  │                     │  │  │  (可滚动, max-h)      │  │ │
│  │  ┌───────────────┐  │  │  │                      │  │ │
│  │  │ 画质选择       │  │  │  └──────────────────────┘  │ │
│  │  └───────────────┘  │  │                            │ │
│  │                     │  │                            │ │
│  │  [开始下载] 按钮     │  │                            │ │
│  └─────────────────────┘  └────────────────────────────┘ │
└──────────────────────────────────────────────────────────┘
```

移动端（< md）自动回退为单列布局。

### 需求 2：SEO/GEO 优化

仅改 `frontend/index.html`，新增：
- `<meta description>` — 产品描述
- `<meta keywords>` — 关键词
- Open Graph 标签（og:title, og:description, og:image, og:url, og:type）
- Twitter Card 标签
- `<meta theme-color>` — 移动端浏览器主题色
- `<link rel="canonical">` — 规范链接
- JSON-LD 结构化数据（WebApplication schema）
- `<meta name="robots">` — 允许索引

### 需求 3：字幕下载 + 思维导图全屏/下载

#### 3a. 字幕下载
字幕 Tab 折叠头旁加「下载 SRT」按钮，前端直接将 segments 转为 SRT 格式触发浏览器下载（纯前端）。

#### 3b. 思维导图全屏 + 下载
导图 Tab 折叠头旁加「全屏」和「下载 PNG」按钮：
- 全屏：overlay 全屏显示 SVG
- 下载：SVG 通过 canvas 转 PNG 下载

## 四、验证方案

1. **UI 紧凑性**: 粘贴 URL → 弹窗左栏视频信息 + 右栏 AI 总结一屏可见
2. **移动端**: 缩小到 768px 以下 → 自动回退单列
3. **SEO**: 查看源码 → 确认 meta/OG/JSON-LD 存在
4. **字幕下载**: 字幕 Tab → 下载 → .srt 文件格式正确
5. **导图全屏**: 导图 Tab → 全屏 → ESC 关闭
6. **导图下载**: 导图 Tab → 下载 → .png 文件正确
7. **已有功能**: 下载、登录、AI 总结/问答正常
